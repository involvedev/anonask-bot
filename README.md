# anonask-bot
## Телеграм бот для обмена анонимными сообщениями.

Что-бы запустить бота заполните файл **.env**, затем пропишите команду **docker compose up --build**.
Для запуска на ПК установите приложение Docker Desktop.

<dr>

## Функции бота
<ul>Отправка и получение анонимных сообщений</ul>
<ul>Просмотр статистики пользователем</ul>
<ul>Кастомизация личной ссылки</ul>

## Функции админ панели:
<ul>Рассылка по всем пользователям</ul>
<ul>Просмотр статистики бота</ul>
